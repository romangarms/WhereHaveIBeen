Set Up OwnTracks

1. Download OwnTracks for iOS or OwnTracks for Android

2. Open the app and tap the (i) button in the top left

3. Tap Settings

4. Change the mode from MQTT to HTTP

5. Set UserID to your username

6. Set Password to your password

7. Set DeviceID to something memorable (e.g., "phone")

8. Set the URL to:
https://mini.romangarms.com/pub
